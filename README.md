# FactionRats
An Instagram bot that automatically posts pictures/videos from other accounts.


# How to use
To set it up, all you need to do is go into the config.txt file, and write the username/password of the instagram account you would like to use, and the directory for the FactionRats folder.
To change the hashtags provided, accounts scraped, or captions given, simply edit the text files (Each line should have one option)

# Dependencies
Instapy-cli,
Instaloader, and
the built-in OS, datetime, random, and itertools modules
